# Track & Field Data Visualization

Interactive visualization of historical athletics results based on the World Athletics all-time top lists dataset.

🌐 **Live Demo**  
👉 https://ricmoccia.github.io/track-and-field-viz/

---

## 📊 About the Project

This project explores and visualizes elite track & field performances using interactive data visualizations.

It includes:

- 📈 Ranking visualizations
- 🥇 Event-based filtering
- 📊 Comparative views
- 🔎 Detailed athlete performance exploration

The dataset is based on historical all-time top lists from World Athletics.
